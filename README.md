# Portfolio

## 概要
[Aik(アイク)](https://github.com/aik0aaac)  
のポートフォリオサイトです。

## 環境構築
※本WebサイトはリポジトリないのHTMLファイルを、jQuery.ajaxメソッドにて部分的に取得しているためサーバー環境からでないとページが見れません。
前提環境: Node.js

- コンソール上で`node server.js`と叩く
- `http://127.0.0.1:3000/`にアクセス
- ポートフォリオが見れます

## 使用した外部ライブラリ等
- Google Fonts+
- jQuery-3.1.1
- Font awesome
- sweetalert
- Fancybox
- Swiper
- imagesloaded
- masonry
